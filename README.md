# MultiUpload

**支持功能**：
- 多任务批量上传；
- 上传状态储存（结合数据库）；
- 进度监听（需具体网络上传框架支持，如okhttp，oss等）；
- 上传取消；
- 上传错误（或强杀）后点击重新上传；
- 可根据实际业务需求改造...
